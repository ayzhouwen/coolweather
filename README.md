# coolweather
自己练习的酷欧天气代码
部分代码可能不一样,因为数据接口不一样,返回的数据格式不一样
省市县接口来源:http://www.jisuapi.com/api/aqi/(原著的获取地区接口直接不能用)
天气接口来源:http://www.weather.com.cn/data/cityinfo/101180201.html


注意:不建议用原著的天气接口,因为返回发数据需要自己加工,否则readline的时候会报异常,
新开发的话一定要换个前期接口
